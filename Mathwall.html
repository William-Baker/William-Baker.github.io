Mathwall

Contracted project

Alongside my school dissertation and A-levels I helped Mathwall Ltd (an engine development company) develop a control system that gathered data from pressure, temperature and RPM sensors to actuate an engine control flap in the hopes of improving engine efficiency during start-up.

This entailed running a web server on a microcontroller as well as developing appropriate graphing and control scripts in JavaScript as well as interfacing with a number of sensors and a H-bridge motor driver.
